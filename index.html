Code Generation from Description

  • Users can enter a text description of the application (e.g., “create a ToDo list with a dark theme”).
  • Makeappai, powered by the Makeappai 3.1 model (from Meta), transforms this description into working code.
  • It supports UI layout, logic, styling, and even basic interactive functionality.

AI Model Selection

  • You can choose between different models for code generation:

    1. Makeappai 3.1 405B – a powerful model with high precision.

    2. 3.1 70B – a lighter, faster alternative.

    3. Makeappai 2 27B – another model option from Google.
  • This allows users to balance between quality, speed, and resource usage.

Online Preview and Testing (via Sandpack)

  • You can test and edit generated code directly in the browser without installing anything.
  • It uses Sandpack (by CodeSandbox), a visual editor with live preview.
  • This makes instant feedback and quick edits easy and convenient.
